# Battleships

## How to run
1. Install [Node.js](https://nodejs.org/en/download/)
2. Install [yarn](https://yarnpkg.com/en/docs/install)
3. `yarn install`
4. `yarn run dev`
5. Run tests with: `yarn run test`
